# TopFiveUser

This component shows the profile picture, username and points of one of the top 5 - 10 users of the week. The level of each user will determine the color bordering their profile picture.

<Props of={TopFiveUser} />

export const Template = (args) => (
  <TopFiveUser
    position={args.position}
    userPicUrl={args.userPicUrl}
    name={args.name}
    userColor={args.userColor}
    points={args.points}
  />
);
