# UserInfo

This component contains all the information about an specific user. It allows you to delete certain information depending on what you want to display.
It also allows you to either show a colored background or a White one

<Props of={UserInfo} />

export const Template = (args) => (
  <UserInfo
    user={args.user}
    backgroundColor={args.backgroundColor}
    textColor={args.textColor}
  />
);
