# Default global styles
The file **`app.scss`** contains a series of default pre-applied styles for the whole application (used on the tag **`body`**).

### Elements displayed centered and sorted vertically by default

```
display: flex;
flex-direction: column;
align-items: center;
```

### Margins and paddings set to 0 by default.

```
margin: 0;
padding: 0;
```

### Font set to the default in vars.scss

```
font-family: $default-font;
```

### Width set to 100% of the screen width on mobiles
```
@media (max-width: $mobile-breakpoint) {
    width: 100vw;
    max-width: 100vw;
    box-sizing: border-box;
}
```
(`box-sizing: border-box` is used to prevent overflowing when using `padding`).
