# TeamDropdown

The `TeamDropdown` component is a dropdown menu that is used to display information about a team or group of people.

## Properties

The component accepts the following properties:

| Property | Description |
|-----------|-------------|
| `title` | The title of the dropdown menu. |
| `data` | The information to be displayed when the menu is expanded. |
| `bgColor` | The background color of the dropdown menu. Default is `#1a3891`. |
| `isExpanded` | The state of the dropdown menu. |
| `onClick` | The function to be executed when clicking on the dropdown menu. |

## Behavior

The dropdown menu changes state when clicked. When expanded, it displays the information provided in the `data` property.

Additionally, the dropdown menu displays a down arrow (<IoIosArrowDown />) when expanded and a right arrow (<IoIosArrowForward />) when closed.
